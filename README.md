# GreenCodeFramework
Full stack open source (MIT) framework for making web apps in PHP+JS

## how to start
1.copy script.ps1 to empty directory
2.run by powershell
```pwsh
. ./script.ps1
start-project
```

remember about enabling execution policy (by default powershell block running scripts)
